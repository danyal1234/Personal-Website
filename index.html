Genealogy Application

CIS2750 Project

GEDCOM is a standard for geneaological data that allows information about families, the individuals those families are comprised of, the relationships between them and other information to be held in a bytestream or file. Geanealogicaol data standards like GEDCOM are commonly used in services such as Ancestry.com or other family tree tracing infrastructures, and as such systems capable of processing this data are required.

Integrated web app with a MySQL database, used for storing individual and family information in a remote DB. With this integration, additional functionality was added to the UI allowing users to send custom queries to the DB and returning the results to the UI.
